# ALL_MultiOmics

This repository contains pipelines and methods to analyse multi-omics data of Acute Lymphoblastic Leukemia (ALL) disease. 

The data comes from the bone marrow samples and PDX Patient derived xenograft) models are created by transplanting human tumor tissue/cells (often obtained through surgery or biopsy) into immunocompromised or humanized mice. This provides a more authentic in vivo model of human tumor biology compared to traditional cell lines.

This project contains 4 data modalities:
- Proteomics data
- Transcriptomics data
- Whole Exome Sequencing data
- Drug screen profile data

Apart from the above data modalities, clinical data is also available like: immunophenotype, diagnosisType, age, sex, cytogenetics, etc.
